== Description ==

The **bzzix_restrictive_categories** plugin allows administrators to control which categories users can post in based on their roles or individual user settings. With this plugin, you can set restrictions on categories and allow only specific users or roles to publish posts within defined categories. Ideal for maintaining control over content categorization.

Features:
- Restrict categories based on user roles or individual users.
- Allow administrators to select allowed categories for restricted users.
- Prevent restricted users from publishing posts in unapproved categories.
- Customizable category restrictions per user.

== Installation ==

To install **bzzix_restrictive_categories**, follow these steps:

1. Download the plugin ZIP file from WordPress.org or from a trusted source.
2. In your WordPress admin dashboard, go to **Plugins > Add New**.
3. Click on the **Upload Plugin** button, select the ZIP file, and click **Install Now**.
4. After installation, activate the plugin through the **Plugins** menu in WordPress.

== Frequently Asked Questions ==

= What is the bzzix_restrictive_categories plugin for? =

The **bzzix_restrictive_categories** plugin helps administrators restrict which categories users can post in, based on their roles or specific settings for individual users. It's useful for managing category access and ensuring content is published in the appropriate sections.

= How do I configure the category restrictions? =

Once activated, the plugin allows you to set category restrictions for users. Go to **Users > Profile** and select the categories users are allowed to post in. You can also control restrictions through user roles from the plugin settings.

= Can I allow users to select categories themselves? =

Yes, the plugin allows you to define which categories a user can select from, but only those that have been authorized. Restricted users will see only the allowed categories when they post.
