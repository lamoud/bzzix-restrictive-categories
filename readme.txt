== Description ==

The **bzzix_restrictive_categories** plugin allows WordPress administrators to control which categories users can post in, based on their roles or individual user settings. This plugin enables category access restrictions, ensuring that only authorized users or roles can publish posts in specific categories. It's ideal for maintaining structured and organized content management within your WordPress site.

Features:
- Restrict category access based on user roles or individual user settings.
- Enable administrators to assign allowed categories for restricted users.
- Prevent restricted users from publishing posts in unauthorized categories.
- Customizable category restrictions for individual users.

== Installation ==

To install **bzzix_restrictive_categories**, follow these simple steps:

1. Download the plugin ZIP file from WordPress.org or from a trusted source.
2. In your WordPress admin dashboard, navigate to **Plugins > Add New**.
3. Click the **Upload Plugin** button, choose the downloaded ZIP file, and click **Install Now**.
4. After installation, activate the plugin through the **Plugins** menu in WordPress.

== Frequently Asked Questions ==

= What is the bzzix_restrictive_categories plugin used for? =
The **bzzix_restrictive_categories** plugin helps administrators restrict which categories users can post in based on their roles or specific settings for individual users. This is particularly useful for managing content and ensuring that posts are categorized correctly.

= How do I configure category restrictions? =
Once the plugin is activated, you can configure category restrictions by going to **Users > Profile**. Here, you can select which categories users are allowed to post in. You can also manage these restrictions via user roles through the plugin settings.

= Can users select their allowed categories? =
Yes, the plugin allows users to select from categories they are authorized to use. Restricted users will only see and be able to choose from categories assigned to them.
